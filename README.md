# VoxelSpace React Application

This is a modern React application built with Vite, Tailwind CSS, and Three.js.

## Prerequisites
- **Node.js**: You need Node.js installed on your computer.

## How to Run

1.  **Open Terminal**: Open a terminal in this directory (`d:\Research Work\Antigravity Stuff\Voxel Website`).
2.  **Install Dependencies**: Run the following command to install all required libraries:
    ```bash
    npm install
    ```
3.  **Start Dev Server**: Run the development server:
    ```bash
    npm run dev
    ```
4.  **Open in Browser**: Click the link shown in the terminal (usually `http://localhost:5173`).

## Project Structure
- `src/App.jsx`: Main layout and logic.
- `src/components/Sidebar.jsx`: Input controls and results display.
- `src/components/Viewport.jsx`: 3D visualization using React Three Fiber.
- `src/index.css`: Tailwind global styles.
